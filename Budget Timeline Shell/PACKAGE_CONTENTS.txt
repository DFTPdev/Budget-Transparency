================================================================================
COMPONENTS OVERVIEW
================================================================================

1. BudgetTimelinePage.tsx
   - Main page with tabs (Timeline + Key Visuals)
   - Handles state management
   - Orchestrates all child components
   - Note: Key Visuals tab requires additional hooks not included

2. TimelineAxis.tsx
   - Visual representation of budget stages over time
   - Shows current stage highlight
   - Interactive tooltips
   - Responsive design

3. StageChips.tsx
   - Clickable chips for each budget stage
   - Shows selected state
   - Triggers milestone filtering

4. MilestoneList.tsx
   - Displays milestones for selected stage
   - Shows dates and explanations
   - Sorted by date

5. WatchNextCard.tsx
   - Shows upcoming events to monitor
   - Highlights important dates

6. SourcesFooter.tsx
   - Lists data sources with links
   - Shows last updated timestamp

================================================================================
HOOKS & UTILITIES
================================================================================

useTimeline.ts exports:
- useTimeline(): Main hook for loading timeline data
- getCurrentStage(): Get current/next stage based on today's date
- getMilestonesByStage(): Filter milestones by stage ID

Features:
- In-memory caching (1 hour TTL)
- Error handling
- Loading states
- TypeScript types

================================================================================
DATA STRUCTURE
================================================================================

budgetTimeline.2024_2026.json contains:
- biennium: FY period (start, end, label)
- stages: 7 budget process stages with date windows
- milestones: Key events for each stage
- watchNext: Upcoming items to monitor
- sources: Data source references
- lastRefreshed: Last update timestamp

INTEGRATION STEPS
================================================================================

1. Install dependencies (npm/yarn/pnpm)
2. Copy folder to your project
3. Place data file in public/data/official/
4. Update import paths in BudgetTimelinePage.tsx
5. Optionally remove "Key Visuals" tab
6. Wrap with MUI ThemeProvider
7. Use the component!
